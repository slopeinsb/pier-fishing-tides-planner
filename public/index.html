How recommendations work

  • Start with a four-hour fishing block centered on each high tide.
  • Favor higher high tides compared with the rest of the day.
  • Favor bigger swings from nearby low tides because moving water often fishes better.
  • Reward calmer wind and manageable wave height using NWS forecast grids.
  • Add a bonus when a window overlaps the hour around sunrise or sunset.
  • Use those conditions to suggest the most likely species mix for each window.
  • Keep the NOAA station visible so you can validate whether a nearby pier is using the right reference point.
